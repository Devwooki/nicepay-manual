# 코드집

## HTTP 상태코드
|http status|설명|
|:---:|:--|
|200|OK, 요청이 정상 처리된 경우|
|400|Bad Request, 입력 파라미터가 없거나 유효하지않은 값일 경우|
|401|Unauthorized, SecretKey 또는 토큰 없거나 인증에 실패한 경우|
|403|Forbidden,|
|404|Not Found, 요청한 리소스가 존재하지 않는 경우|
|500|Server Error, 나이스페이먼츠 서버에 오류가 발생한 경우|
